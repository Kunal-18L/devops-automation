# DevOps Automation Project

## Technologies
- AWS
- Terraform
- Ansible
- Git
- GitHub

## Project Structure
- terraform/
- ansible/

## Features
- Create EC2 using Terraform
- Configure EC2 using Ansible
- Install Java
- Install Git
- Install Maven
- Install DOcker
- Ansible Roles
- Variables
- Handlers
- Templates

## How to Run

### Terraform

terraform init
terraform apply

### Ansible

ansible-playbook -i inventory site.yml
